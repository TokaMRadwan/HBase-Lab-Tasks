# HBase Lab Tasks

This repository contains practical HBase exercises and mini-projects demonstrating NoSQL database concepts and HBase Shell operations.

## Topics Covered

- Namespace Management
- Table Creation and Modification
- Data Insertion (put)
- Data Retrieval (get, scan)
- Row Filtering
- Count Operations
- Table Administration
- Product Inventory System
- Log Analysis System

## Technologies Used

- Apache HBase
- Hadoop Ecosystem
- Linux (CentOS)
